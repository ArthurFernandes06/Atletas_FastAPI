API de Cadastro de Atletas

  Projeto simples em FastAPI utilizando SQLAlchemy e SQLite para cadastro de atletas, categorias e centros de treinamento.

Funcionalidades

  -Criar, listar e deletar categorias

  -Criar, listar e deletar centros de treinamento

  -Criar atletas vinculados a categoria e centro de treinamento

Tecnologias

  -Python

  -FastAPI

  -SQLAlchemy

  -SQLite
